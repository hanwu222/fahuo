# 成品账号自助购买系统

这是一个基于纯前端技术的数字商品自动发货系统。

## 📁 项目文件

| 文件 | 说明 |
|------|------|
| `index.html` | 商品展示页面（买家使用） |
| `order.html` | 订单查询页面（买家使用） |
| `admin.html` | 后台管理页面（店主使用） |
| `style.css` | 样式文件 |
| `app.js` | 前台功能逻辑 |
| `admin.js` | 后台管理逻辑 |
| `config.js` | 配置文件 |
| `alipay-qr.jpg` | 支付宝收款码 |

---

## 🚀 快速开始（本地测试）

### 方法1：直接打开
双击 `index.html` 在浏览器中打开即可使用。

### 方法2：使用本地服务器（推荐）
```bash
# 使用 Python
python -m http.server 3000

# 或使用 Node.js
npx serve .
```
然后访问 `http://localhost:3000`

## 📖 使用流程

### 买家购买流程
1. 访问网站首页 → 看到商品信息和收款码
2. 扫码付款 ¥45
3. 填写联系方式和交易号后4位 → 提交订单
4. 保存订单号，等待发货
5. 在订单查询页面查看状态，发货后即可获取账号

### 店主发货流程
1. 访问 `admin.html` → 输入管理密码登录
2. 首次使用：点击"批量上传文件"→ 添加库存
3. 收到付款通知后 → 在后台找到对应订单 → 点击"发货"
4. 系统自动分配一个库存给该订单

```javascript
  // 商品信息

    name: '成品账号',
    price: 45,
    currency: 'CNY'
  }
};
```

## 🌐 免费部署到互联网

### 选项1：Vercel（推荐）
1. 注册 [Vercel](https://vercel.com)（免费）
2. 安装 Vercel CLI：`npm i -g vercel`
3. 在项目目录运行：`vercel`
4. 按提示完成部署
5. 获得一个公开网址如 `xxx.vercel.app`

### 选项2：GitHub Pages
1. 创建 GitHub 仓库
2. 上传所有文件
3. Settings → Pages → 选择 main 分支
4. 获得网址如 `username.github.io/repo`

### 选项3：Netlify
1. 注册 [Netlify](https://netlify.com)（免费）
2. 拖拽项目文件夹上传
3. 自动获得公开网址

## ⚠️ 重要提示

1. **修改默认密码**：部署前务必修改 `config.js` 中的管理密码
2. **本地存储限制**：当前版本使用浏览器本地存储，数据仅保存在本地
3. **后续升级**：如需云端数据库，可配置 Supabase（实施计划中已说明）
4. **收款确认**：由于是个人收款码，需要手动确认收款后再发货
